一、功能详述

二、求助者的权利与义务

  1. 求助者负责提供项目需求详细资料以及相关的技术指标。
  2. 求助者托付大神开发的软件在签订协议之后，如需增加其他功能，必须以书面形式告知大神以及毕设帮平台，大神有权酌情增加费用。
  3. 求助者需配合大神完成项目的环境配置运行以及相关技术支持。
  4. 求助者有权在项目期内，要求大神出示项目进度证明，实时监管。

三、大神的权利与义务

  1. 大神只对求助者提出的协议内要求进行开发。
  2. 大神所开发软件是自行研发的，不侵犯任何第三方的合法权益。
  3. 大神将严格按照求助者提出的各项技术需求及指标，进行软件的开发设计。
  4. 大神开发的软件不得含有病毒、反动以及违反国家法律法规规定的内容。
  5. 大神交付软件后，必须向求助者提供相关技术支持以及售后维护。

四、求助者验收标准

  1. 求助者验收时，必须严格按照协议规定的功能点，不得无故添加功能。
  2. 大神所开发的软件符合协议要求的所有功能点以及技术指标即为合格。
  3. 大神提出验收申请之后，求助者应在三日之内进行验收，超过十五日不验收，视为验收合格。
  4. 求助者验收不合格，可由双方协商解决，如有争议，可向毕设帮客服申请仲裁。

五、费用结算方式

  1. 该项目求助者付给大神人民币元整。（平台服务费10%）
  2. 甲乙双方签订合同当日，求助者将项目开发款项交付给毕设帮平台监管。
  3. 大神交付软件由求助者验收合格之后，毕设帮平台交付大神人民币元整。
  4. 大神收取报酬之后，免费为求助者维护软件三个月。

六、违约责任

  1. 由于求助者中途更改软件开发要求而导致大神工作不能按时完成，大神不负任何责任，并有权向求助者提出延期要求，延期时间由双方协商决定。
  2. 由于大神单方面原因导致项目延期，求助者有权向大神提出索赔，具体额度由双方协商决定，但最高不超过合同中求助者支付费用总金额的10%。
  3. 由于大神单方面原因导致项目未能按合同要求完成，求助者有权向毕设帮申请全额退款。

七、争议解决

  • 本合同履行过程中，如发生任何争议，双方应本着友好合作的精神协商解决，如若协商未果，可向毕设帮平台申请仲裁。

八、合同更改与终止

  • 任何一方更改或终止协议，必须以书面形式通知另一方以及毕设帮平台，由毕设帮平台本着公平公正的原则，按照协议说明，妥善处理钱款。

九、附则

  1. 协议签订之后未尽事宜，由双方协商一致，对本协议有关条款进行更改或者补充时，应以书面形式予以确认。
  2. 本协议一式两份，双方各持一份，具有同等法律效力。
  3. 本协议更改或者补充附录，具有同等法律效力。
